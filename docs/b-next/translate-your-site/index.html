安装

使用者提示

Next用户可阅读此文档进行体验，请阅读下一章的后续教程

  • Windows
  • MacOS
  • Ubuntu/Debian
  • Centos

推荐系统 Windows10 | Windows11

其他windows版本请安装 Google Chrome / Edge

必要数据库 Redis-x64-5.0.14.1.msi

必要环境 Node.js@18.20.3

推荐使用nvm-setup.exe对Node.js进行版本切换

Node.js 版本必须大于16.14.0且双数版本

必要工具 Git Download for Windows

git必须全部点击默认next，其他地址可能会造成git损坏

推荐系统 MacOS12 | MacOS14

非最新的系统不保证其成功性

必要工具 Git 一般 MacOS 自带

git --version # 校验git版本，未安装会提示下载常用工具包

必要数据库 Redis-7.4.2

需要进行编译，请自行翻阅相关文章

必要工具 brew

# 下载
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
# 添加环境环境
echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.bash_profile
# 添加环境环境
echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zshrc
# 刷新环境
source ~/.bash_profile && source ~/.zshrc

必要工具 npm

# 安装
brew install nvm
# 添加环境
echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.bash_profile
# This loads nvm
echo '[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"' >> ~/.bash_profile
# This loads nvm bash_completion
echo '[ -s "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"' >> ~/.bash_profile
# 添加环境
echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.zshrc
# This loads nvm
echo '[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"' >> ~/.zshrc
# This loads nvm bash_completion
echo '[ -s "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"' >> ~/.zshrc
# 刷新环境
source ~/.bash_profile && source ~/.zshrc

必要环境 Node.js@18.20.3

# 安装
nvm install 18.20.3
# 使用
nvm use 18.20.3
# 校验版本
node -v
npm -v

示例系统 Ubuntu 24.04 LTS / Debian 12.0 X86 2H2G

  • 确保apt包是最新的
sudo apt update
sudo apt install wget -y
sudo apt install curl -y
  • 安装NVM
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.0/install.sh | bash
  • 添加NVM环境环境

~/.bashrc

echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.bashrc
echo '[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"' >> ~/.bashrc
source ~/.bashrc # 刷新环境
nvm -v # 版本
  • 安装 Node.js@18.20.3
# 安装
nvm install 18.20.3
# 使用
nvm use 18.20.3
# 校验版本
node -v
npm -v
  • 安装git
apt install git
  • 安装redis
apt install build-essential tcl -y
# 进入目录包目录
cd /usr/local
# 下载
wget  "http://download.redis.io/releases/redis-6.2.13.tar.gz"
# 解压
tar zxvf "redis-6.2.13.tar.gz"
# 进入redis目录
cd redis-6.2.13
# 编译
make
make install
# 启动
./src/redis-server --daemonize yes
  • 安装chromium
sudo apt install chromium-browser
